# Logo Setup Instructions

## Adding Your Company Logo

1. **Prepare your logo file:**
   - File format: PNG (recommended) or SVG
   - Recommended size: 120px width × 80px height maximum
   - Transparent background recommended
   - File name: `logo.png`

2. **Add logo to the project:**
   - Place your logo file in: `public/logo.png`
   - Replace the existing placeholder file

3. **Logo will automatically:**
   - Resize to fit the container (max 120×80px)
   - Maintain aspect ratio
   - Display in the top-right corner of the dashboard
   - Have a glass-effect container with border
